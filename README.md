# Harvest Life Care Website

A clean, organized static website for Harvest Life Care.

## Project Structure

```
HarvestLife/
├── index.html          # Home page (221KB)
├── css/                # Stylesheets (29 files)
├── js/                 # JavaScript files (18 files)
├── images/             # Images and graphics (29 files)
├── webfonts/           # Font Awesome webfonts (9 files)
```

## Features

- ✅ Clean folder structure
- ✅ All resources local (works offline)
- ✅ Font Awesome icons working
- ✅ Elementor animations enabled
- ✅ No external dependencies
- ✅ No tracking scripts

## Usage

Simply open `index.html` in your web browser to view the website.
